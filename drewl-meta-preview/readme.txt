The Drewl Meta Preview plugin will display how your content (post or page) will show on different social sharing platforms.

== Description ==
This plugin uses [Open Graph](https://ogp.me/) tags to display a preview of your content on on different social sharing platforms. To be able to properly work it will need a tier plugin that will generate these Meta tags or you can manually specify them in your theme\'s header template.

== Installation ==
**Install it manually**

1. Download the .zip file from the [Wordpress Plugin database](https://wordpress.org/plugins/drewl-meta-preview/)
1. Extract it on you local machine
1. Upload the plugin folder to the `wp-content/plugins/` folder of your web server.
1. Go to your WordPress dashboard, menu `Plugins`, and activate the `Drewl Meta Plugin`

**Install it from whithin your WordPress admin pannel**

1. Go to the `Plugins > Add New` menu of your WordPress admin panel
1. Search for `Drewl Meta Preview`
1. Install and Activate `Drewl Meta Preview` from the search result

== Frequently Asked Questions ==
= Why does the plugin need Open Graph tags? =
Open Graph tags are now a common standard for social sharing platforms to fetch information about your web page (Title, description, image...) and display them on their feeds.
